• git init- Initialize a new git repository in the current directory

  • git add file- Adds a specific file to the staging area

  • git commit -m "message"- creates a new commit with the changes in the staging area and specific area and specifies the commit message inline

  • git push origin main- pushes local commits to the remote repository

  • git clone- this clones a repository from a remote server to your local machine

  • ls- list of all files in current being tracked by git in your repository

  • git log --all- displays the commit history of all branches

  • git status- Shows the current state of your repository,including tracked and untracked files,modified files,and branch information

  • git fetch- Retrieves change from a remote repository,including new branches and commit

  • git pull origin main- Fetches changes from the remote repositoryand merges them into the current branch
